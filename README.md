# 🌦 Weather App

A simple and modern **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data from the **OpenWeatherMap API**.

The application allows users to enter a **city name** and retrieve current weather information such as temperature, humidity, wind speed, pressure, and precipitation.

---

## 🚀 Features

- Search weather by **city name**
- Displays **real-time weather data**
- Shows important weather attributes:
  - 🌡 Temperature
  - 💧 Humidity
  - 💨 Wind Speed
  - 🧭 Wind Direction
  - 🌫 Atmospheric Pressure
  - 🌧 Precipitation

- **Hero section background**
- **Glassmorphism transparent weather card**
- Uses **Fetch API with Async/Await**
- Weather data displayed using **DOM Manipulation**
- Responsive and modern UI

## 🔑 API Used

This project uses the **OpenWeatherMap Current Weather API**.

API Endpoint:

```
https://api.openweathermap.org/data/2.5/weather
```

Documentation:

https://openweathermap.org/current

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/weather-app.git
```

### 2️⃣ Navigate to the project folder

```
cd weather-app
```

### 3️⃣ Get your API Key

1. Go to **https://openweathermap.org**
2. Create a free account
3. Generate an **API Key**

---

### 4️⃣ Add API Key

Open **script.js** and replace:

```
const apiKey = "YOUR_API_KEY";
```

with your actual API key.

Example:

```
const apiKey = "abcd123456789xyz";
```

---

## ▶️ Run the Project

Simply open **index.html** in your browser.

Or use **Live Server (VS Code extension)**.
